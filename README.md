# Beginner Gemini Chatbot

This project is a simple Python chatbot that uses the Gemini API.

## Setup
1. Copy `.env.example` to `.env`.
2. Put your Gemini API key in `.env`.
3. Install dependencies:
   pip install -r requirements.txt
4. Run:
   python main.py

Note: Keep your API key secret. Do not commit `.env` to version control.
